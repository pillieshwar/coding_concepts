Find : Local Maximum & Global Maximum
Maximum Subarray Problem : The maximum subarray problem is the task of finding the largest possible sum of a contiguous subarray, within a given one-dimensional array A[1â€¦n] of numbers.

import math
arr = [-2,1,3,4,-1,2,1,-5,4]
local_max = 0
global_max = -math.inf

for i in range(len(arr)):
    local_max = max(arr[i] + local_max,arr[i])
    if(local_max>global_max):
        global_max = local_max
print(global_max)
